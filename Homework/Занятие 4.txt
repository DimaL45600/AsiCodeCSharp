1. Метод для нахождения максимума из трех чисел:

Напишите метод MaxOfThree, который принимает три целых числа в качестве параметров и возвращает максимальное из них. Затем вызовите этот метод и выведите результат на экран.

2.  Возврат нескольких значений через out:

Напишите метод DivideAndRemainder, который принимает два целых числа и возвращает результат деления и остаток через параметры с использованием ключевого слова out.


3. Обратная строка через рекурсию:

Напишите рекурсивный метод ReverseString, который принимает строку и возвращает её в обратном порядке.
