Создайте класс банковский счет с полями: номер счета, владелец, баланс. Реализуйте методы для пополнения счета, снятия средств, проверки баланса. Добавьте логику для проверки достаточности средств при снятии.

Создайте класс "Библиотека"
Реализуйте поиск книг по автору, жанру, году издания. Добавьте возможность сортировки книг по различным критериям.

3. Расширение функционала библиотеки:

Расширим задачу про библиотеку. Создадим класс Читатель с полями: имя, фамилия, номер читательского билета, список взятых книг. Реализуем метод для выдачи книги читателю и возврата книги в библиотеку.
