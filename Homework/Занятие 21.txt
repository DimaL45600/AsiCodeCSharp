Сервер клиентских заказов

Реализовать приложение, в котором клиент отправляет данные о заказе на сервер, а сервер сохраняет его и подтверждает получение.

Создайте класс Order с полями:
	int Id (идентификатор заказа),
	string ProductName (название продукта),
	int Quantity (количество),
	DateTime OrderDate (дата заказа).
Клиентское приложение отправляет объект класса Order на сервер.
Сервер принимает объект, сохраняет его (например, в списке List<Order>), и отправляет подтверждение клиенту.
