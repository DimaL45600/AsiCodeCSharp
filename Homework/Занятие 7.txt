1.  Система расчета налогов

Описание задачи:
Разработайте систему расчета налогов для различных типов доходов. Используйте перечисления для описания типов доходов, глобальные конфигурации налоговых ставок, а также методы для расчета налогов.

Требования:

Используйте enum для описания типов доходов: Salary, Business, Investments.
Создайте класс Income с readonly полями для типа дохода и суммы.
Реализуйте статический класс TaxConfig, который содержит глобальные налоговые ставки для каждого типа дохода.
Добавьте статические методы для расчета налогов на основе типа дохода и ставки.
Реализуйте статическое поле для хранения общей суммы собранных налогов.

2. Система управления библиотекой

Описание задачи: Разработайте систему управления библиотекой, которая позволяет пользователям брать и возвращать книги. Книги могут быть в разных форматах и жанрах (используйте enum), а конфигурация библиотеки и информация о её состоянии должны быть доступны глобально (используйте static и readonly).

Требования:

Используйте enum для описания форматов книг (например, Hardcover, Paperback, Ebook, Audiobook) и жанров (например, Fiction, NonFiction, Mystery, ScienceFiction).
Создайте класс Book с readonly полями, такими как Title, Author, Format, и Genre.
Реализуйте класс LibraryConfig, который будет содержать глобальные настройки для библиотеки (например, максимальное количество книг в каталоге), используя static и readonly поля.
Добавьте статические методы для управления библиотекой: выдача книги, возврат книги, проверка доступности.
Реализуйте статическое поле для отслеживания общего количества взятых книг.
