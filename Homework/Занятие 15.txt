Учёт заявок на ремонт
Создайте базу данных для учёта заявок на ремонт с таблицами Customers (Клиенты), Technicians (Техники) и ServiceRequests (Заявки на обслуживание).
Задачи:
Создайте таблицу Customers с полями ID, Name, ContactInfo.
Создайте таблицу Technicians с полями ID, TechnicianName, Specialty.
Создайте таблицу ServiceRequests для заявок на ремонт с полями CustomerID, TechnicianID, RequestDate, Status.
Создайте форму для добавления клиентов и техников
Реализуйте метод для создания новой заявки и назначения техника, а также создайте форму для вывода информации по заявквм.
Создайте метод для обновления статуса заявки (например, "В ожидании", "В работе", "Завершено").
Сделайте форму меню с кнопками для перехода по другим формам формам
Напишите метод, который возвращает все заявки, выполненные техником за определённый период и выводит данные например в ListBox.
